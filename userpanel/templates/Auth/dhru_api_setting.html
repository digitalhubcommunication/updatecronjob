Setup Instructions

To enable API automation features to run, make sure you set up a cronjob to run every 5 seconds, 10 seconds. Create the following CronJob using CRONJOB.CO.UK. How to set up a Cronjob in the panel:

  • Set up separate cron jobs as listed below:
  • API Settings
  • Cron Settings
  • Separate Cron For Send and Get: OFF
  • SetCronJob.com (Recommended)

After clicking "Save", the Cronjob will be automatically added to all the services on your domain, and your Fastcronjob will work.
